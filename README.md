## 🎨 About
RT CarMenu is designed to provide a **clean and modern interface** for vehicle interactions.

- Smooth user experience  
- Minimalistic and responsive design  
- Easy to use and integrate  

---

## ✨ Features
- Menu opens smoothly from the **left side**  
- Clean and modern UI  
- Multiple **color themes** available  
- Simple and intuitive layout  
- Lightweight and optimized  

---

## ⚙️ Customization
You can customize the menu in the `config` file:

- Change **color themes**
- Adjust visual settings
- Modify menu behavior

Make it match your server style perfectly.

---

## 📦 Installation
1. Download or clone this repository  
2. Add the resource to your `resources` folder  
3. Configure settings as needed  
4. Start/restart your server
